=====================================================================================================
命令行自动安装脚本: mysql_install.bat的简单用法

确定一个完整的版本号，比如：5.7.37, 8.0.28
访问网址：
https://mirrors.sohu.com/mysql/MySQL-<主版本号>/ (主版本号为5.1, 5.6, 5.5之类)
找到winx64.zip结尾的文件名，比如：mysql-5.7.37-winx64.zip， 中间的串：5.7.37就是完整的版本号.

可以选定是32位的， 还是64位的，32位用win32, 64位的用winx64, 默认使用winx64.

命令行用法：
mysql_install.bat <fullversion> [arch] [ROOTDIR] [server_port]
参数:
     fullversion,  必须指定, 如5.6.14, 5.1.72, 4.1.22, 5.5.34, 6.0.11-alpha等等
     arch, 可选，不指定时，为winx64, 有效值可以是win32或者winx64
     ROOTDIR, 可选，安装的最终根目录，默认值为c:\mysql， 不用带后边的"\".此目录必须事先存在或创建好
     server_port, 可选，默认值为3306.
     
最终生成的服务名为: mysql-<主版本号>

如果你想卸载，也非常简单，先用sc delete mysql-<主版本号>
接着进到%ROOTDIR%\%MYSQL_HOME%, 直接删除整个目录就可以了. 非常方便。

使用时，也请尊重个人劳动。注明出处。
